# Branch logos

Each branch has three files

- logo-name.svg
- logo-name-flat.svg
- logo-name.png

The `logo-name.svg` is the workable .svg file with the text editable.

The `logo-name-flat.svg` has the text turned into paths, so that they can be imported into non-inkscape tools or tools that do not have the font installed. This version will also show up properly in the GitHub preview

The `logo-name.png` file is a rasterized version of the vector file.
