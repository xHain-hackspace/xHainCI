# Branch logos

Each branch has three files

- branch-name.svg
- branch-name-flat.svg
- branch-name.png
- branch-name-design.svg

The `branch-name.svg` is the workable .svg file with the text editable.

The `branch-name-flat.svg` has the text turned into paths, so that they can be imported into non-inkscape tools or tools that do not have the font installed. This version will also show up properly in the GitHub preview

The `branch-name.png` file is a 500px rasterized version of the vector file. Please use https://tinyjpg.com to optimise the file!

In `branch-name-design.svg` you will find the colors separated, hex codes and other design elements.

In the folder `affinity-designer-files` you will find the `.af` files Affinity uses.
